# Security Policy

## Reporting Security Issues

If you discover a security vulnerability, please email us at **madhurmarketing.ai@gmail.com** instead of using the public issue tracker.

Include the following details:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

We will respond within 48 hours and work with you to fix it before making it public.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Security Best Practices

When using AI Voice Agent:

1. **Keep API Keys Safe** - Never commit `.env` files or keys to GitHub
2. **Use HTTPS** - Always use encrypted connections
3. **Limit Permissions** - Use the minimum needed permissions
4. **Update Regularly** - Keep dependencies up to date
5. **Monitor Access** - Watch for unusual activity
